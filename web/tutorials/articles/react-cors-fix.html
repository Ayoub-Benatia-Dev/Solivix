حل مشكلة CORS في React.js

تعتبر مشكلة CORS من المشاكل الشائعة التي تواجه مطوري React.js عند التعامل مع واجهات API. في هذا الشرح سنتعرف على أسباب المشكلة وكيفية حلها بطرق مختلفة.

ما هي مشكلة CORS؟

CORS هي آلية أمان تفرضها المتصفحات لمنع الطلبات من مصادر مختلفة. تحدث المشكلة عندما يحاول تطبيق React الوصول إلى API من نطاق مختلف.

مثال على المشكلة 
// الكود المسبب للمشكلة
fetch('https://api.example.com/data')
  .then(response => response.json())
  .catch(error => console.error('CORS Error:', error));

طرق الحل

1. إعداد Proxy في React

// في package.json
{
  "proxy": "http://localhost:5000"
}

2. إعداد الخادم

// في Express.js
app.use(cors());
